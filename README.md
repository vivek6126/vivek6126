# 🌟 Featured Project

# 🏨 TripNest — Full-Stack Hotel Booking Platform

A production-ready hotel booking platform inspired by Airbnb and Booking.com.

> **📷 Add a screenshot of TripNest here for maximum impact.**

**Stack:** Next.js 16 · React · TypeScript · Tailwind CSS · shadcn/ui · PostgreSQL · Supabase · Stripe

### Highlights

* Built a complete booking flow with server-side booking conflict detection to prevent overlapping reservations.
* Integrated Stripe Checkout with webhook verification so bookings are confirmed only after successful payment.
* Implemented Supabase Authentication with Server-Side Rendering and Row Level Security (RLS) to secure user data.
* Designed a responsive user experience including wishlists, dashboards, reviews, profile management, and booking history.
* Deployed on Vercel with preview deployments for every update.
